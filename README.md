# 智能写作助手

基于智谱 AI 的智能写作辅助工具，支持多种写作模式和流式输出。

## 功能特性

### 核心功能
- ✨ **6 种写作模式**：文章续写、内容改写、内容扩展、内容总结、邮件撰写、文案生成
- 🎯 **提示词优化**：AI 自动优化用户输入，提升生成质量
- 📊 **参数调节**：可调节创意度（temperature）和输出长度
- 🌊 **流式输出**：实时显示生成内容，提升用户体验
- 📝 **Markdown 渲染**：支持 Markdown 格式的内容展示
- 📚 **历史记录**：自动保存生成历史，支持查看和删除
- 📋 **一键复制**：快速复制生成的内容

### 界面设计
- 🎨 左右分栏布局，清晰直观
- 💫 现代化 UI 设计，使用 Tailwind CSS
- 📱 响应式设计，支持移动端
- 🎭 流畅的动画效果

## 快速开始

### 1. 获取 API Key

访问 [智谱 AI 开放平台](https://open.bigmodel.cn/) 注册并获取 API Key。

### 2. 配置 API Key

1. 打开应用
2. 点击右上角的 "⚙️ 设置" 按钮
3. 输入您的智谱 AI API Key
4. 点击 "保存"

### 3. 开始使用

1. 选择写作模式（续写、改写、扩展等）
2. 在输入框中输入您的内容
3. 调整创意度和输出长度（可选）
4. 点击 "✨ 生成内容" 按钮
5. 在右侧查看实时生成的内容
6. 点击 "📋 复制" 按钮复制结果

### 📚 历史记录

**适用场景**：
- 查看之前的生成内容
- 重用之前的提示词和参数
- 管理和组织创作历史

**功能特点**：
- ✅ 自动保存每次生成的内容
- ✅ 显示生成时间（相对时间）
- ✅ 显示使用的参数（创意度、长度）
- ✅ 点击加载历史内容和参数
- ✅ 单条删除或批量清空
- ✅ 最多保存 100 条记录
- ✅ 本地存储，隐私安全

**保存内容**：
1. 写作模式
2. 原始提示词（完整内容）
3. 生成结果
4. 创意度参数
5. 输出长度参数
6. 生成时间

**使用方法**：
- 点击历史记录卡片：加载该记录的所有内容和参数
- 点击删除图标：删除单条记录（需确认）
- 点击"清空"按钮：清空所有历史记录（需确认）

## 参数说明

### 高级设置面板

点击 "⚙️ 高级设置" 可以展开/折叠高级参数控制面板。

### 创意度（Temperature）
- **范围**：0.0 - 1.0
- **默认值**：0.7
- **调节方式**：滑块
- **说明**：
  - 较低值（0.0-0.3）：输出更加确定和保守，适合需要准确性的场景
  - 中等值（0.4-0.7）：平衡创意和准确性，适合大多数场景
  - 较高值（0.8-1.0）：输出更加多样和创意，适合创意写作

### 输出长度（Max Tokens）
- **范围**：100 - 2000 tokens
- **默认值**：1500 tokens
- **调节方式**：滑块
- **说明**：
  - 100-500：简短回复，适合摘要和简答
  - 500-1500：中等长度，适合一般内容
  - 1500-2000：较长内容，适合详细文章

**注意**：1 token ≈ 1.5 个中文字符，因此 1500 tokens 约等于 1000 个汉字。

### 快速预设

高级设置面板提供三个快速预设按钮：

- **🛡️ 保守**：创意度 0.3，长度 1000 tokens - 适合正式文档、技术文章
- **⚖️ 平衡**：创意度 0.7，长度 1500 tokens - 适合大多数场景
- **🎨 创意**：创意度 0.9，长度 2000 tokens - 适合创意写作、营销文案

## 注意事项

1. **API Key 安全**：API Key 存储在浏览器本地，请勿在公共设备上使用
2. **网络连接**：需要稳定的网络连接以访问智谱 AI API
3. **使用限制**：请遵守智谱 AI 的使用条款和配额限制
4. **数据隐私**：历史记录仅保存在本地浏览器，不会上传到服务器

## 常见问题

### Q: 历史记录保存在哪里？
A: 历史记录保存在浏览器的 localStorage 中，仅存储在本地，不会上传到服务器。

### Q: 历史记录会占用多少空间？
A: 最多保存 100 条记录，通常占用几 MB 空间。如果超过限制，会自动删除最旧的记录。

### Q: 如何备份历史记录？
A: 目前历史记录存储在浏览器本地。建议定期复制重要内容到其他地方保存。

### Q: 清除浏览器数据会丢失历史记录吗？
A: 是的，清除浏览器的 localStorage 数据会删除所有历史记录。

### Q: 生成失败怎么办？
A: 请检查：
1. API Key 是否正确配置
2. 网络连接是否正常
3. 是否超出 API 配额限制

### Q: 如何清除历史记录？
A: 可以逐条删除历史记录，或清除浏览器的 localStorage 数据。

### Q: 如何清除历史记录？
A: 点击历史记录面板右上角的"🗑️ 清空"按钮可以清空所有记录，或点击单条记录的删除图标删除特定记录。

### Q: 支持哪些浏览器？
A: 支持所有现代浏览器（Chrome、Firefox、Safari、Edge 等）。
