## 🚀 如何使用

### 方法 1: 在线访问（推荐）
1. 访问 GitHub Pages（如果已启用）
2. 或者下载到本地使用

### 方法 2: 克隆到本地
```bash
# 克隆仓库
git clone https://github.com/ChrisQuella/AI-Writing-Assistant.git

# 进入目录
cd AI-Writing-Assistant

# 打开应用
# 双击 index.html 或使用本地服务器
```

### 方法 3: 下载 ZIP
1. 访问仓库页面
2. 点击 "Code" -> "Download ZIP"
3. 解压后双击 `index.html`

## 📝 快速开始

1. **设置 API Key**
   - 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/)
   - 获取 API Key
   - 在应用中点击 "⚙️ 设置" 保存

2. **开始使用**
   - 选择写作模式
   - 输入内容
   - 点击 "✨ 生成内容"

3. **查看文档**
   - `README.md` - 完整功能说明
   - `QUICK_START.md` - 快速上手指南
   - `TEST_CHECKLIST.md` - 测试清单
